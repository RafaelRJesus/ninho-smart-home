# Definition of Ready e Definition of Done

## Definition of Ready

- História possui objetivo de negócio.
- Critérios de aceite estão testáveis.
- Dependências estão identificadas.
- Design ou wireframe está disponível quando necessário.
- Contrato de API está definido.
- Riscos estão registrados.
- Massa de teste está planejada.

## Definition of Done

- Código implementado e revisado.
- Critérios de aceite atendidos.
- Testes unitários aprovados.
- Testes de API e integração aprovados.
- Testes de interface aprovados.
- Sem vulnerabilidade crítica ou alta.
- Logs, métricas e tracing implementados.
- Documentação atualizada.
- Deploy em QA realizado.
- Evidências anexadas.
- Aprovação funcional registrada.
- Rollback validado.
